Removes js, css and ajax behaviors from Contact Form 7 that may slow the site

== Description ==

Removes js, css and ajax behaviors from Contact Form 7 that may slow the site.

This plugins is now supported and maintained by <a href="https://www.fivestudio.it/" target="_blank">Fivestudio</a>.

== Installation ==

1. Upload plugin directory to the `/wp-content/plugins/` directory. You can also upload the zip the file in the file upload input.
2. Activate the plugin through the 'Plugins' menu in WordPress

== Usage ==

This plugin does NOT require any configuration or usage. It simply removes some js, css and ajax behaviors that may slow the site.
